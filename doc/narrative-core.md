# Narrative core (SSOT)

**AK.skill** is a pattern pack for when public ideas shift from “here is my repo” to “here is the **pattern** your agent can instantiate.” In Cursor, start from the **`ak-skill`** entry skill (`/ak-skill`) to pick the right checklist path. It distills a repeatable **idea release path**: surface the discomfort of a real workflow change, name the new abstraction (optionally with a bounded metaphor or era label), pair the claim with a minimal proof—sometimes an **irreducible single-file core** or a **scoreboard-driven loop**—hand off an agent-readable idea file when attention spikes, and always weld hype to a single concrete discipline (including eval honesty and paradox where needed). For long arcs, a **stack year review** shape syncs the field without replacing deeper essays. Use these skills for posts, talks, READMEs, or memos—not as impersonation of any individual, but as a **literacy template** for technical leadership in public.
